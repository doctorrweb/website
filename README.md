# doctorrWeb Website

## Usage

- Rename the template.env to .env and fill it with the correct values

## Install Dependencies

````
npm install
````

## FRONT END 

> Generate the bundles 

`````
npm run watch
`````

> Run the App

```````
npm run dev
```````


## BACK END 

> Generate the bundles 

`````
npm run build
`````

> Run the App

```````
npm start
```````
